Создай новый проект React, используя команду npx create-react-app. 
Убедись, что ты находишься в нужной папке перед выполнением команды. 
После создания проекта открой его в редакторе кода и создай три компонента: `Header`, `Main` и `Footer`. 
Каждый компонент должен быть оформлен в отдельном файле и находиться в папке components.

Импортируй созданные компоненты в файл `App.js` и оберни их в обычный `<div>`, 
добавив класс wrapper. Убедись, что все компоненты правильно отображаются на странице. 
Если возникнут ошибки, попробуй самостоятельно их найти и исправить. 
Это поможет тебе лучше понять, как работает React.

Применяй CSS Grid для стилизации своего приложения. 
Определи основные стили для `wrapper`, чтобы задать ширину и цвет фона.
Используй свойства **CSS Grid** для задания имен областей и расположения компонентов на странице. 
Не забудь добавить отступы между элементами, чтобы интерфейс выглядел аккуратно и удобно для восприятия.
